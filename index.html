Features

Built for Resilient Communities

BlueMesh combines solar energy, Bluetooth mesh networking, and a dedicated Android app to keep barangay communities connected.

BLE Mesh Networking

Decentralized device-to-device communication using Bluetooth Low Energy. Messages hop between nodes, extending signal range via flooding relays.

Solar-Powered Nodes

Relay nodes are powered by 50,000mAh solar power banks, ensuring uninterrupted 24/7 off-grid operations.

No Internet Required

Operates completely offline via Bluetooth. Designed for typhoons and outages when cellular and internet infrastructure fail.

Android Mobile App

A native Android application built with Kotlin and Jetpack Compose for low-latency text messaging.

Low-Latency Buffering

Integrates LZ4 payload compression, dynamic MTU fragmentation, and an SQLite store-and-forward offline queue.

Disaster Resilient

Uses standalone ESP32D microcontrollers acting as signal relays. Weatherproof nodes serve as community signal boosters.

How It Works

Simple, Powerful, Off-Grid

Solar-powered relay nodes and your Android phone are all it takes to build a resilient communication network.

01

Deploy Solar Relay Nodes

Place weatherproof, solar-powered Bluetooth mesh relay nodes at strategic points across your barangay. They charge themselves and run 24/7.

02

Install the App

Download and install the BlueMesh APK on any Android device. No account creation needed — just open the app and you're ready.

03

Discover Nearby Peers

BlueMesh scans for nearby devices using Bluetooth Low Energy. Other users within range automatically appear in your peer list.

04

Send Messages

Start chatting with real-time text messaging. Messages hop through relay nodes and nearby devices, reaching people across the barangay.

About the Project

Communication for Every Barangay

BlueMesh is an open‑source, peer‑to‑peer Bluetooth mesh messaging app for Android. It enables devices to communicate without internet or cellular networks, using solar‑powered ESP32 relay nodes to extend range.

Features include proximity distance detection, opt‑in location sharing, secure PIN lock, automatic auto‑lock, offline message queue with SQLite, LZ4 compression, fragmented MTU packetizer, and full support for Android 15/16 edge‑to‑edge UI.

Evaluation Results

Validated by User Trials

Purposive user acceptance testing with 20 IT, Engineering, and non-technical respondents was conducted to validate usability and multi-hop relay success.

90%

Multi-Hop Relay Success

90% of tested transmission runs successfully propagated across multi-hop relay nodes (65% working seamlessly, 25% with minor propagation delay).

3.88

Usability Score (Mean)

Overall rating of 3.88 (Agree) on a 5-point Likert scale. UI Ease of Use and Message Transmission Speed scored highest at 4.15/5.00.

75%

Emergency Use Case

75% of evaluators identified Emergency/Disaster Response as the primary intended use case, strongly validating the off-grid design focus.
