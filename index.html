Features

Built for Resilient Communities

BlueMesh combines solar energy, Bluetooth mesh networking, and a dedicated Android app to keep barangay communities connected.

📡

BLE Mesh Networking

Decentralized device-to-device communication using Bluetooth Low Energy. Messages hop between nearby devices, extending reach up to 300 meters per hop.

☀️

Solar-Powered Nodes

Relay nodes are powered by solar panels, ensuring uninterrupted operation during grid outages and power failures. Self-sustaining energy for 24/7 resilience.

📴

No Internet Required

Works completely offline via Bluetooth. Designed for typhoons, power outages, remote barangays, and areas where cellular infrastructure is unavailable.

📱

Android Mobile App

A user-friendly Android application for real-time, low-latency text messaging. Just open and start communicating with nearby users instantly.

🛡️

Weatherproof Hardware

Relay nodes are housed in weatherproof enclosures designed to withstand harsh Philippine weather — including typhoons, heavy rain, and extreme heat.

🌪️

Disaster Resilient

Purpose-built for emergency and disaster response. When typhoons knock out cell towers and power grids, BlueMesh keeps your community connected.

How It Works

Simple, Powerful, Off-Grid

Solar-powered relay nodes and your Android phone are all it takes to build a resilient communication network.

01

Deploy Solar Relay Nodes

Place weatherproof, solar-powered Bluetooth mesh relay nodes at strategic points across your barangay. They charge themselves and run 24/7.

02

Install the App

Download and install the BlueMesh APK on any Android device. No account creation needed — just open the app and you're ready.

03

Discover Nearby Peers

BlueMesh scans for nearby devices using Bluetooth Low Energy. Other users within range automatically appear in your peer list.

04

Send Messages

Start chatting with real-time text messaging. Messages hop through relay nodes and nearby devices, reaching people across the barangay.

About the Project

Communication for Every Barangay

BlueMesh is an open‑source, peer‑to‑peer Bluetooth mesh messaging app for Android. It enables devices to communicate without internet or cellular networks, using solar‑powered ESP32 relay nodes to extend range.

Features include proximity distance detection, opt‑in location sharing, secure PIN lock, automatic auto‑lock, offline message queue with SQLite, LZ4 compression, fragmented MTU packetizer, and full support for Android 15/16 edge‑to‑edge UI.

🔗 View BlueMesh Repository on GitHub
